# Запуск pyjobkit taskboard в Portainer

1. Убедитесь, что у вас есть доступ к интернету из контейнеров (для git и pip).
2. В Portainer создайте новый стек и вставьте содержимое вашего `docker-compose.yml` (уже обновлённого).
3. Нажмите Deploy the stack — Portainer скачает официальный образ python:3.13-slim, выполнит установку зависимостей и запустит FastAPI-приложение.
4. После запуска сервис будет доступен на порту 8000 (например, http://<ваш-host>:8000/).

## Локальный запуск (альтернатива)

```bash
docker compose up taskboard
```

## Примечания
- Все данные и задания хранятся в памяти, при перезапуске контейнера история очищается.
- Для production рекомендуется собирать и публиковать собственный образ в реестр.
- Если нужен доступ к приватному репозиторию, используйте переменные окружения для передачи токена или SSH-ключа.
